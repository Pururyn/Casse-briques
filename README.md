# 🧱 Breakout

A simple Breakout game built in **C++** with **SFML**.

Move the paddle to keep the ball in play and destroy all the bricks to win.

## Controls

| Key | Action |
|-----|--------|
| `←` / `→` | Move paddle |
| `Escape` | Quit |

## Requirements

- Visual Studio
- SFML 2.x
